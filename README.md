# NiceHttp
A small and simple OkHttp wrapper to ease scraping. 
Mostly for personal use.

Featuring:
- Document scraping using jsoup
- Json parsing using jackson
- Easy functions akin to python requests
- Easy cache

Currently, lacking: 
- Requests beyond get, post and put
